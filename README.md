Esse é um material feito sem fins lucrativos para ajudar na confecção de um site para a entidade estudantil Onda Elétrica.

O objetivo é atrair uma maior quantidade de acessos ao site e por consequência atriar mais público nos espaços futuros do Onda elétrica

Caso vá mexer no Repositório, evite trocar de lugar os arquivos para não quebrar partes do projeto.

Metas estabelecidas:

1. Ajeitar a página de equipe.
  1.1 Colocar em categorias(Membros e Probatório).
  1.2Preencher os cars com nome,foto e semestre de ingresso de cada.
   
3. Ajeitar a página de atividades
  2.1 Separar as atividades nbas 4 categoriais
  2.2 Colocar ícones clicáveis associados a cada atividade
  2.3 Cada atividade levar a uma página exclusiva dela que menciona projetos e etc.
  2.4 Traçar um plano de como melhor organizar isso
   
3.Buscar novas ideias para adicionar mais coisas ao site
  3.1 Sugestão pessoal -> Aba com foto dos nossos experimentos, como funcionam e vídeo em prática.
